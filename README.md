I want to also be able to add and remove rows and execute SQL commands

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/sqlite-table-editor.git
cd sqlite-table-editor
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
